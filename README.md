## Procesamiento de Datos a Gran Escala
## Por Ana Sofía Arboleda

Este repositorio presenta una colección de cuadernos desarrollados para la resolución de problemas mediante Python y el análisis estructurado de datos.

El objetivo del repositorio es evidenciar habilidades en programación, documentación y organización de proyectos orientados al procesamiento de datos.


## Estructura del repositorio

- `Proyectos/`  
  Contiene los cuadernos en Python correspondientes a los talleres prácticos.
  Cada cuaderno incluye:
  - Tema
  - Conceptos
  - Descripción del objetivo del taller
  - Desarrollo de los ejercicios
  - Conclusiones finales


## Competencias desarrolladas

- Programación en Python
- Diseño y manipulación de Bases de Datos NoSQL para Big Data.
- Uso de diferentes tecnologías y técnicas de almacenamiento y procesamiento de Big Data para entender su aplicación en las organizaciones.
- Procesamiento y análisis de información.
- Escritura de código legible.
- Presentación clara de resultados.
